# Note Taking App

A simple note-taking application built using Python and `tkinter`.

## Features

- **Write and Edit Notes**: Use the text area to create and edit notes.
- **Save Notes**: Save your notes to a text file (`.txt` format).
- **Open Notes**: Open existing text files and view their contents.

## Installation

1. Make sure you have Python installed on your system. You can download it from [python.org](https://www.python.org/).
2. Save the `note_taking_app.py` script to your local machine.

## Usage

1. Run the script using Python:
   ```bash
   python note_taking_app.py
